Welcome to DotDaht.com!

Dot Daht aims to be a community built website, where the community votes on the next HTML page to add. This is just the very beginning so there isn't a lot here yet, but I hope there soon will be.
This page will be expanding as I think of stuff, so make sure to check back every once and a while.
If you have suggestions, please post them, in the sub-reddit https://reddit.com/r/dotdaht

INFORMATION AND RULES

PROCESS
Every Friday, at 5PM MST, the top ten(10) highest voted submissions on the sub-reddit from the last week will be gathered and included in a poll that will last two(2) days until Monday 12AM. At that point, the highest voted submission will be added to the website automatically, and the process repeats.

RULES
1. All pages must be added by community vote.
2. Submissions must adhere to the submission rules. All submissions in violation will be removed from consideration or deleted from the site.
3. If a page with the same path of another page is voted in, it will overwrite the existing file.
4. index.html and faq.html are exempt from being overwritten.
5. Anything is up for discussion in regards to how this site operates.
6. I reserve the right to change or break said rules at any time.
7. Sorry about the last one, but this is still a work in progress. I need that flexability.

SUBMISSIONS
1. Submissions must consist of a single file that contains only HTML, CSS, and Javascript.
2. Submissions can be no larger than 5MB in size.
3. Submissions must be hosted on either github.com or gitlab.com
4. Including external Javascript files is forbidden.
5. Any page considered malicious if forbidden.

SUBMISSONS

Submissions must be made in the Dot Daht sub-reddit and must adhere to the rules. Please refer to the sub-reddit sidebar for further submission information.

REPORTING

If you find something wrong with the site, or a page that doesn't adhere to the guidelines, please make a post reporting it on the Dot Daht sub-reddit.
